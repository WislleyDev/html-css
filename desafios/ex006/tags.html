Tags em HTML

Em HTML5.podemos incerir os elementos da lista a seguir.Passe o mouse sobre os termos para descubrir as tags utilizadas em cada caso.

  • Parágrafos
    • Use a tag <p>
  • titulos
    • Use as tags <h1>...<h6>
  • imagens
    • Use a tag <img>
  • abreviações
    • Use a tag <abbr>
  • código fonte
    • Use a tag <code>
  • listas numeradas
    • Use as tags <ol> e <li>
  • listas com demarcadores
    • Use as tags <ul> e <li>
